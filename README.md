# keyboard-cheatsheet
A cheatsheet for keyboard shortcuts

## Windows

| Shortcut                                 | Description                      |
| ---------------------------------------- | ---------------------------------|
| WIN + 1,...,0                            | Open window 1,...0 from taskbar  |
| WIN + T                                  | Focus windows taskbar            |


## Firefox

| Shortcut                                 | Description                      |
| ---------------------------------------- | -------------------------------- |
| CTRL + T                                 | New Tab                          |
| CTRL + F4                                | Close Tab                        |
| CTRL + TAB (+ SHIFT)                     | Switch Tab                       |
|                                          |                                  |
